# BCI Hackey Sac — Game Plan

## Status

LIVE

## MVP

Complete. A falling ball cycles through 3 colors (blue, orange, purple). Match the color
with the correct click to kick it back up. Practice mode and Game mode both functional.
Analytics tracking per-finger accuracy, hold duration, and reaction time across sessions.

## Long-Term Vision

- Rainbow sequence bonus: hit all three colors in order for a streak multiplier or power kick
- Sound effects: satisfying audio feedback on correct kick, wrong click, game over
- Visual polish pass: particle effects, screen shake on wrong click, smoother animations
- Difficulty presets: Easy / Medium / Hard as quick-select instead of manual slider tuning
- Cross-game analytics: session data feeds into the launcher's global BCI profile
